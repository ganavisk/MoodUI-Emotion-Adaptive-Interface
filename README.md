# MoodUI – Emotion Adaptive Interface

## Overview
MoodUI is a user interface that dynamically changes theme and color based on the user's mood selection. This project demonstrates adaptive UI and personalized user experience.

## Features
- Mood-based theme switching
- Dynamic interface feedback
- Clean minimal UI design
- Interactive buttons

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in a browser
3. Select a mood and observe the UI change

## Future Improvements
- AI-based emotion detection
- Personalized recommendations
